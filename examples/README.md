# Examples

This directory contains examples demonstrating how to use the `lookup` library.

## Contents

*   Basic: Demonstrates `Get`, `Set`, and `Exists` operations.
*   Create: Shows how to use `Create` to initialize nested structures.
*   Parsing: Illustrates type conversion and custom parser hooks.

## Running the Examples

You can run each example from the root of the repository.

### Basic Usage

```bash
go run examples/basic/main.go
```

### Creating Nested Structures

```bash
go run examples/create/main.go
```

### Type Parsing

```bash
go run examples/parsing/main.go
```
